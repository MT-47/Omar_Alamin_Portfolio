Project 01

Smart 4S Battery Management System

Overview

Designed the full hardware architecture, schematic, and multi-layer PCB for a 4S lithium cell battery management system. The system provides comprehensive protection against overcurrent, overvoltage, and thermal faults, ensuring safe and reliable battery operation in demanding applications.

Implemented an ESP32-based real-time monitoring system that continuously reads cell voltages, temperatures, and load current. Live data is displayed on-screen and transmitted via IoT protocols, while protective shutdowns are triggered automatically when thresholds are exceeded.

The design was validated through multiple prototyping iterations, with each board tested under load to verify protection thresholds and monitoring accuracy before finalizing the production-ready layout.
